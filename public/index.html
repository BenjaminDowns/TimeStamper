Time Stamper

     unix date <== language
         ||           /\
         ||           ||
         ||           ||
         \/           ||
      language ==> unix date

How to:

1) Go to https://time-stamper.herokuapp.com/some-date-here

2) Time Stamper responds with a JSON object containing both the unix stamp and your natural language date.

Examples:

Request: https://time-stamper.herokuapp.com/Feb%2026%202015

Response: { unix: 1424905200, natural: "Thursday, February 26, 2015" }

Request: https://time-stamper.herokuapp.com/November%2028,%201962

Response: { unix: -223866000, natural: "Wednesday, November 28, 1962" }
